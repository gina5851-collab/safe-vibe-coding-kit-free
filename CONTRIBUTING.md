# Contributing

Thank you for your interest in improving Safe Vibe Coding Kit.

This project is designed for non-coders, creators, educators, and small business owners who use AI coding tools such as Claude Code, Cursor, GitHub, and Vercel.

## Ways to contribute

You can help by:

- Improving documentation
- Adding beginner-friendly examples
- Suggesting safer AI coding prompts
- Reporting confusing instructions
- Translating guides into Korean or English
- Adding checklists for common non-coder workflows

## Good first contributions

- Add screenshots for GitHub setup
- Improve the pre-deploy checklist
- Add examples of safe Claude Code prompts
- Add examples of safe Cursor prompts
- Translate key guides

## Community expectations

Please keep contributions clear, practical, and beginner-friendly.

Do not include private data, API keys, customer information, payment information, or proprietary project files.
